# `KIP.JS`

Kip.Js is an unopinionated Javascript Framework that helps developers develop <a href="http://nodejs.org" target="_blank">Node.js</a> server side applications faster with a maintainable architecture and fully customization of the framework to meet your needs. It is built with <a href="http://www.typescriptlang.org" target="_blank">TypeScript</a> hence it is type safe and combines multiple paradigms of programming.
